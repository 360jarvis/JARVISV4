# Jarvixx 2.0 Design System

## Brand Direction

Jarvixx must feel like a luxury SaaS platform: clean, calm, premium, fast, and trustworthy.

## Core Colors

- Champagne Gold: primary accent
- Black: sidebar, premium contrast, high-value CTA moments
- Ivory: app background
- White: cards and content surfaces
- Soft Gold: highlights and selected states
- Muted Brown Gray: secondary text

## UI Rules

- No heavy/bold visual clutter.
- Use compact professional spacing.
- Use cards for grouped information.
- Use in-app modals and drawers only.
- Avoid external popup windows.
- Tables should feel dense but readable.
- Default table pagination: 25 rows.
- Buttons must be real actions or clearly marked Coming Soon.
- No fake success states.

## Components Required

- App Shell
- Sidebar
- Page Header
- KPI Card
- Data Table
- Search Bar
- Filter Bar
- Status Badge
- Action Button
- Modal
- Drawer
- Empty State
- Timeline
- Calendar Panel
- Customer Profile Header
- AI Assistant Panel

## Formatting

- Dates: MM/DD/YYYY.
- Use day of week where it helps scheduling.
- Phones: (XXX) XXX-XXXX.
- Currency: USD.
- All customer-facing copy must be clear and professional.

## Accessibility

- Use semantic page structure.
- Maintain readable contrast.
- Inputs must have labels or accessible names.
- Interactive elements must be keyboard reachable.
